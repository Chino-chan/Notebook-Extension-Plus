# Notebook-Extension-Plus 🥰

This is a modified version of "Notebook Extension" by Cohee1207
https://github.com/SillyTavern/Extension-Notebook

## What was added

1. Now notes are divided either per "Character Card" or "Global".
3. Inside "Character card" you can choose to store notes either for the card itself, or for specific chat files within the card.
4. Added some text-color tools & copy to clipping tools for mobile users

## How to use

1. Install using the link: `https://github.com/Chino-chan/Notebook-Extension-Plus`
2. Find a new option "Open Notebook-Plus" in the wand menu or use the `/notebookplus` slash command.
3. Create a new note and start writing.

### How to build

1. Clone the repo
2. Run `npm install`
3. Run `npm run build`
4. Minimized plugin file will be in the `/dist` folder
